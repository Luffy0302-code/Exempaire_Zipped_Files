To use this application, use this command (when you're in the folder EZF_tool.exe is located. Or if you're not, run this command: cd C:\path\to\your\folder), (EZF_tool.exe -c ".\Your folder" The name of your EZF.EZF)
And to decompress the file: (EZF_tool.exe -x The name of your EZF.EZF ".\Your Folder").

In simple terms:
To compress: EZF_tool.exe -c ".\Your folder" The name of your EZF.EZF
To decompress: EZF_tool.exe -x The name of your EZF.EZF ".\Your Folder"
